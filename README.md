# 🏙 القنايات 3D — مدينة أونلاين

مدينة القنايات (محافظة الشرقية) مبنية ثلاثية الأبعاد على **شبكة الشوارع الحقيقية** من OpenStreetMap، مع نظام أونلاين متعدد اللاعبين: تسجيل دخول، حركة بالشخصية، شات، مكالمات صوتية، وتبادل بيانات التواصل.

## 📦 المتطلبات
- [Node.js](https://nodejs.org) (نسخة 18 أو أحدث)
- إنترنت لتحميل مكتبة Three.js وخط Cairo (مرة واحدة عند فتح الصفحة)

## ▶️ التشغيل
افتح Terminal/CMD جوه مجلد المشروع ونفّذ:

```
npm install
npm start
```

بعدها افتح المتصفح على:
- من نفس الجهاز: **http://localhost:4173**
- من أي جهاز تاني على نفس الشبكة (واي فاي): **http://IP-الجهاز:4173** — السيرفر بيطبع العنوان ده لما يشتغل.

## 🎮 طريقة اللعب
| الحاجة | الطريقة |
|---|---|
| تسجيل الدخول | اكتب اسمك واختار شكل ولون شخصيتك (الواتس/الفيس/الرقم اختياري) |
| الحركة | الأسهم أو WASD (مع Shift للجري) |
| الانتقال السريع | اسحب شخصيتك وارميها في أي مكان — أو دبل كليك على الأرض |
| الكلام مع لاعب حقيقي | اضغط عليه ← 💬 رسالة / 📞 مكالمة صوتية / 📇 طلب التواصل |
| قصص أهل البلد | الشخصيات اللي فوقها اسم أصفر = شخصيات خيالية، اضغط عليها تقرأ قصصها المتشابكة |
| ليل/نهار | زرار 🌙 فوق |

## 🔒 الخصوصية
بيانات الواتس/الفيس/الرقم **مش بتظهر لحد** غير لما حد يطلب التواصل **وتوافق بنفسك** — ساعتها بس بتتبادلوا البيانات (الطرفين ياخدوا بيانات بعض).

## 🎙 ملاحظة عن المكالمات الصوتية
المتصفحات بتسمح بالمايكروفون على `localhost` عادي. لو بتلعبوا من أجهزة مختلفة على الشبكة بـ `http://IP:4173` المتصفح هيمنع المايك (سياسة أمان) — الحلول:
1. شغّل اللعبة عبر HTTPS (مثلًا وراء خدمة زي ngrok / Cloudflare Tunnel)، أو
2. في كروم افتح `chrome://flags/#unsafely-treat-insecure-origin-as-secure` وضيف `http://IP-السيرفر:4173`.

الشات وكل حاجة تانية شغالين عادي من غير أي إعداد.

## 🗂 هيكل المشروع
```
qanayat-city/
├── server.js          ← السيرفر (ملفات + WebSocket للاعبين)
├── package.json
├── public/
│   ├── index.html     ← اللعبة كاملة (Three.js)
│   └── data/osm.json  ← شوارع القنايات الحقيقية من OpenStreetMap
└── README.md
```

## 💾 نقل المشروع لجهاز تاني
انسخ مجلد `qanayat-city` كله (ممكن من غير `node_modules`)، وعلى الجهاز الجديد: `npm install` ثم `npm start`. خلاص.
